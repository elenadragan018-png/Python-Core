## 🗂️ Structura cursului (10 lecții)

| Lecția | Conținut | Nivel Bloom |
|--------|----------|-------------|
| L1 | Colecții avansate — tuple, dict, set, comprehensions | Apply, Analyze |
| L2 | Șiruri și expresii regulate — metode str, f-strings, modulul `re` | Apply |
| L3 | Funcții avansate — lambda, map/filter, recursivitate, decoratori | Analyze |
| L4 | Module și pachete — os, math, datetime, json, importlib | Understand, Apply |
| L5 | **Recapitulare L1–L4** — quiz cumulativ de 6 itemi cu feedback | Evaluate |
| L6 | OOP Introducere — clase, obiecte, `__init__`, atribute, metode | Understand, Apply |
| L7 | OOP Avansat — moștenire, polimorfism, `__str__` / `__repr__` | Analyze, Evaluate |
| L8 | Excepții și debugging — try/except/finally, raise, logging | Apply, Evaluate |
| L9 | **Proiect aplicativ** — manager de contacte (JSON + OOP + excepții) | Create |
| L10 | **Test final** — 10 itemi pe taxonomia Bloom, prag 70%, certificat PDF | Sinteză |

**Studii de caz incluse:** recursivitatea prin PageRank simplificat (L3), arhitectura unui joc cu clase Player/Enemy/Game (L7), aplicație CLI pentru gestiunea datelor cu persistență JSON (L9).

## 🚀 Utilizare

**Online:** deschideți pur și simplu linkul GitHub Pages de mai sus.

**Local / offline:** descărcați `index.html` și deschideți-l în orice browser modern (Chrome, Edge, Firefox, Safari). Conexiunea la internet este necesară doar pentru fonturi și pentru generarea certificatului PDF.

**Pentru exersarea codului:** Python 3.12+ instalat local (IDLE, VS Code) sau un mediu online.
